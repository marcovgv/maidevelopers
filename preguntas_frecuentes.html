Preguntas Frecuentes

Podríamos definir el termino Dominio como el nombre de un sitio WEB en internet. Todos los sitios WEB tienen una dirección única que se compone de números, los principales objetivos del dominio es facilitar la memorización de la dirección de un sitio WEB y que las personas te visiten, por esta razón un Dominio es el nombre que se asocia a tu sitio WEB. Ejemplo: www.suempresa.com

El Correo electrónico Empresarial es una dirección de Correo electrónico como cualquier otro pero contiene el nombre comercial de una empresa. Ejemplo: contacto@suempresa.com

  • Una cuartilla máximo de información
  • Descripción del proyecto o del Servicio o Producto que ofrecen
  • Formas de contacto: Teléfono Fijo, Teléfono Móvil, Correo Electrónico
  • Dirección de Redes Sociales que desean publicar en la Página WEB
  • Fotos adjuntas en el correo de sus servicios o productos
  • Logotipo en grande formato .png con fondo transparente
  • Nombre del dominio que desea contratar: ejemplo miempresa.com, o nosotros podemos sugerirle uno en base a su proyecto, empresa o servicio. Puede ser terminación .com .com.mx o .mx
  • Algún otro material que usted crea necesario para el desarrollo de la Página WEB
